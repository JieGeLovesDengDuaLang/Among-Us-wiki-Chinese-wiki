# 欢迎来到 Among Us中文维基百科！
## 关于游戏
 Among Us是一款由Innersloth开发的团队合作游戏。在游戏中，船员阵营必须合作找到伪装者（亦可译为“冒名顶替者”，下用“内鬼”来对"impostor"这个词进行翻译）亦或是完成任务来取胜；而内鬼则需要对船员进行击杀，接着使用技能来混淆视听。当船员和内鬼的比值≤1时，内鬼胜利。
